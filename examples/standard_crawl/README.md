# Standard Crawl Example

This folder is a trimmed copy of the official Swumsuit `standard_crawl.zip` example project from `https://www.swum.org/swumsuit/`.

Included files:

- `body_geometry.dat`
- `joint_motion.dat`
- `analysis_settings.dat`
- `comment.dat`

These are enough to run [swum_stick_figure.py](/C:/Users/geraw/SwimMe/SwamSuit/swum_stick_figure.py:1).

Example:

```bash
python swum_stick_figure.py --project-folder examples/standard_crawl
```
